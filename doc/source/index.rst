pycns documentation
===================

Reader and tools for analysing CNS monitor data

Installation
------------

.. code-block:: bash

   git clone https://github.com/samuelgarcia/pycns.git
   cd pycns
   pip install -e .


Update:

.. code-block:: bash
   
   cd pycns
   git clone pull origin main
